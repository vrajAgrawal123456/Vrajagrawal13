# 🍌 Banana Calculator

A fun and colorful calculator built with HTML, CSS, and JavaScript.

## Features

- Basic arithmetic: addition, subtraction, multiplication, division
- Percentage calculation
- Expression display (shows full equation)
- Backspace and clear buttons
- Responsive banana-themed design

## Project Structure

```
banana-calculator/
├── index.html    # Main HTML structure
├── style.css     # Banana-themed styling
├── script.js     # Calculator logic
└── README.md     # This file
```

## How to Run

Just open `index.html` in any browser — no setup needed!
